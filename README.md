# Codegraph Viewer

Web client to visualize, browse, and analyze relationship networks in CodeGraph databases generated by the [CodeGraph](https://colbymchenry.github.io/codegraph/) parser.

**Visit:** [https://codegraph-viewer.jonnallex.site](https://codegraph-viewer.jonnallex.site/)

## Local Setup & Deployment

Run the application instantly in a production-ready Nginx container on port `8201`:
```bash
docker compose up -d --build
```
Then navigate to: `http://localhost:8201`
